## 📊 Risk Scoring System

### Classification Tiers

```
┌─────────────────────────────────────┐
│  0-30   │ ✅ SAFE       │ Green    │
│  31-60  │ ⚠️ SUSPICIOUS │ Yellow   │
│  61-100 │ 🚫 MALICIOUS  │ Red      │
└─────────────────────────────────────┘
```

### Score Calculation

| Security Check | Points | Condition |
|----------------|--------|-----------|
| No HTTPS | +30 | HTTP protocol |
| Phishing Keywords | +20 each | Per keyword found |
| Excessive Subdomains | +20 | >3 subdomains |
| Suspicious TLD | +25 | .xyz, .top, etc. |
| Long URL | +15-25 | >100 characters |
| Special Characters | +15 | Excessive use |
| IP Address | +40 | Direct IP URL |
| Blacklisted | +100 | Known threat |
